# 分段对话·改

将 LLM 长文本按 `\n` 切割为多段短消息，配合可调延迟算法模拟真人输入节奏。

## 功能

- **换行分段**：按 `\n` 将文本切分为多段发送
- **非文本组件策略**：图片、At、表情等可独立配置为 单独/跟随上段/跟随下段/嵌入
- **智能回复**：`enable_reply` 总开关 + `enable_smart_reply`（仅被插话时回复）
- **文本清理**：分段前正则清理 + 分段后句尾符号清理
- **拟真延迟**：线性、对数、随机、固定四种策略
- **范围控制**：可设仅 LLM 回复分段或全部分段，支持黑白名单

## 配置项

通过 AstrBot 管理面板配置，主要分组：

| 分组 | 说明 |
|---|---|
| 基础设置 | 群聊分段开关、分段范围、最长字数、黑白名单 |
| 分段规则 | 清理首尾空行 |
| 文本清理 | 前置正则清理、句尾符号清理 |
| 回复与组件 | 回复开关、智能回复、图片/At/表情策略 |
| 发送延迟 | 策略选择及各策略参数 |
